# Travelo


## Back-end

commands to start the backend 
```
cd Back-end
npm install
```
create .env file 
```
MONGO_URL = "mongodb+srv://Ash1:Ash1@cluster0.pbef3.mongodb.net/myFirstDatabase?retryWrites=true&w=majority" //replace with your username and password
JWT_SEC='adfdf'
PASS_SEC='dfefs'
```

```
npm run dev
```
